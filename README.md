# Freelance Photography Retoucher Template

A premium modern HTML5 website template for a "Freelance Photography Retoucher (High-End Fashion)" brand with a luxury editorial visual style.

## Features
- Minimalist, luxury, ultra-modern design
- 13 Pre-built Pages
- Dark & Light Mode Support
- RTL Language Support
- Advanced Animations (AOS, GSAP, Swiper)
- Mobile-first, fully responsive structure

## Directory Structure
- `assets/css` - Contains main styles, dark mode overrides, and RTL styles
- `assets/js` - Main JavaScript and Dashboard interactions
- `assets/images` - Image assets
- `pages` - All HTML pages

## Technology Stack
- HTML5
- Tailwind CSS (via CDN for easy customization)
- AOS (Animate on Scroll)
- Swiper JS
- Phosphor Icons

## How to edit
All pages use Tailwind CSS classes. You can edit the layouts by modifying the HTML classes directly. Custom CSS variables for fonts and base styles are located in `assets/css/style.css`.
